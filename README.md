# Vide

Vide is short cut for [Vi]m I[DE]. This is a vim config project.

## install

### Manual

cd ~
git clone https://github.com/mrytsr/vide

git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
cd .vim && vim +BundleInstall +qa
